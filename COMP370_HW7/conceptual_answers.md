1.	Why is the difference between found and designed data?
Found Data: Data previously collected and are not organized for the data science project that we are doing.
Designed Data: Data collected specifically for the data science project. 
2.	What are the two primary challenges that necessitate sampling when collecting data?
Practical constraints: It's impossoble to collect all the data due to storage limitations, so we have to organize the data. 
Representativeness and bias: The data might contain redundant information. Sampling needs to be done. 
3.	Why are website owners more likely to be upset about collecting data using scrapers than using APIs?
APIs return clean structured data in consistent formats like json. APIs also manage server load.
